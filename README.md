# simple-image-process-app

単純な画像処理を入れたwebAPI
 
# 使い方
最初にUnet_landscape.ipynbを動かしてください。

その後、app.pyを動かしてください。
 
# 機能
64×64ピクセルの風景画の絵を滑らかにする
